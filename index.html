Welcome to HTML Slide

This is a simple application built on top of backbone and jquery for making presentation slides on HTML.If you don't have MS PowerPoint , you can use this. Download all the files from the source. Open the index.html in browser. You will see presentation with some sample slide in html page.

Prerequisites

  • jQuery
  • backbone
  • underscore
  • handlebars

Please have little knowledge on these frameworks for easy understanding.

HTMLSlide API's and Template

API

**createSlide **: This will create & returns backbone view for the slide.

Parameters (as JSON object):

  • slideClass - css classname for slide main div,
  • header - title appears on top of the slide,
  • sectionClass - css classname for the content div of the slide,
  • slideTemplate - HTML template for the slide ,
  • content - content of the slide ( HTML, text, image)

returns : Backbone view of a Slide

createSlidePoints : This will create a list of points on the slide

Parameters:

  • pointsArr - Array of strings ( points ),
  • SlideView - Object returned with this 'createSlide',

returns : undefined

Template

    <div class="slide {{slideClass}}">
        <div class="slide-header"></div>
        <header>{{header}}</header>
        <section class={{sectionClass}}></section>
        <span class="counter"></span>
    </div>

Add this template inside a script tag and use it while creating slide.

How to create Slide

Load the htmlslide.js as part of the prerequisites in your page. It will return you the HTMLSlide class available in your window context. Create a instance of it by passing a html element present in your dom. Then create the slides with the createSlide API. Follow the sample code below 

//Slide 1
var htmlSlide = new HTMLSlide($('.slides'));
var slide1 = htmlSlide.createSlide({
    slideClass : "current",
    header : "", 
    sectionClass : "middle", 
    slideTemplate : slideTemplate,
    content : '<p>HTML Slide Application using Backbone</p>'
});
