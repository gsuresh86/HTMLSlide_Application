Welcome to HTML Slide

This is a simple application built on backbone and jquery for making presentation slides on HTML.If you don't have MS PowerPoint , you can use this.

Prerequisites

  • jQuery
  • backbone
  • underscore
  • handlebars

Please have little knowledge on these frameworks for easy understanding.

HTMLSlide API's and Slide Template

API

createSlide : This will create & returns backbone view for the slide.

Parameters (as JSON object):

  • slideClass - css classname for slide main div,
  • header - title appears on top of the slide,
  • sectionClass - css classname for the content div of the slide,
  • slideTemplate - HTML template for the slide ,
  • content - content of the slide ( HTML, text, image)

returns : Backbone view of a Slide

createSlidePoints

Parameters:

  • pointsArr - Array of strings ( points ),
  • SlideView - Object returned with this 'createSlide',

returns : undefined

Template

How to create Slide

Load the htmlslide.js as part of the prerequisites. It will return you the HTMLSlide class to the window object. Create a instance of it by passing the html element present in your dom. Use the 

//Slide 1
var htmlSlide = new HTMLSlide($('.slides'));
var slide1 = htmlSlide.createSlide({
    slideClass : "current",
    header : "", 
    sectionClass : "middle", 
    slideTemplate : slideTemplate,
    content : '<p style="font-size:35px>HTML Slide Application</p><p>Using Backbone & Jquery</p>'
});
